# SOP Pengawas Proyek Dari Awal Hingga Serah Terima

Pengawas proyek memiliki peran penting dalam memastikan pekerjaan berjalan sesuai gambar kerja, spesifikasi teknis, target mutu, jadwal proyek, serta standar keselamatan kerja.

Panduan ini membahas tahapan SOP Pengawas Proyek mulai dari persiapan pekerjaan, pengawasan lapangan, quality control, monitoring progress, pengendalian material, dokumentasi proyek hingga serah terima unit perumahan.

---

## Tahapan SOP Pengawas Proyek

### 1. Persiapan Sebelum Pekerjaan Dimulai

- Memahami gambar kerja
- Memahami spesifikasi teknis
- Memahami jadwal proyek
- Menentukan target mutu proyek
- Memahami metode kerja kontraktor

### 2. Pengawasan Harian Lapangan

- Memastikan pekerjaan sesuai gambar
- Mengontrol jumlah tenaga kerja
- Mengontrol penggunaan material
- Memastikan pekerjaan aman
- Mendokumentasikan progres harian

### 3. Quality Control (QC)

- Pemeriksaan struktur
- Pemeriksaan arsitektur
- Pemeriksaan finishing
- Pemeriksaan utilitas
- Pemeriksaan detail pekerjaan

### 4. Monitoring Progress Pekerjaan

- Membandingkan target progress dan realisasi
- Mengidentifikasi hambatan proyek
- Menentukan tindakan korektif
- Mengurangi risiko keterlambatan

### 5. Pengendalian Material dan Tenaga Kerja

- Memastikan material sesuai spesifikasi
- Mengontrol jumlah material
- Mengukur produktivitas tenaga kerja
- Menghindari pemborosan

### 6. Dokumentasi dan Pelaporan

- Monitoring proyek
- Evaluasi proyek
- Audit proyek
- Persiapan serah terima

### 7. Persiapan Serah Terima Unit

- Pemeriksaan QC akhir
- Penyelesaian punch list
- Dokumentasi lengkap
- Verifikasi pekerjaan selesai
- Persetujuan internal

---

## Manfaat SOP Pengawas Proyek

✅ Pekerjaan lebih terkontrol

✅ Mutu proyek lebih terjaga

✅ Progress lebih mudah dipantau

✅ Risiko kesalahan lebih kecil

✅ Serah terima unit lebih lancar
